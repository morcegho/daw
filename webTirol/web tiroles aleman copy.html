Guía de Estilo - Grupo de Folk Tirolés

1. Identidade Visual

A nosa web combina elementos tradicionais do folk tirolés con un deseño moderno e dinámico. A paleta de cores escolleita representa:

  • Vermello Tradicional (#E63946): Representa a paixón e a energía
  • Azul Mariño (#1D3557): Evoca a serenidade das augas
  • Dourado (#FFD700): Simboliza a riqueza da nosa herdanza cultural

2. Elementos de Interface

Os nosos compoñentes seguen estas directrices:

  • Bordos redondeados (border-radius) para unha aparencia suave e moderna
  • Efectos hover sutís para mellorar a interactividade
  • Sombras suaves para dar profundidade
  • Gradientes que evocan as paisaxes alpinas

3. Animacións e Transicións

Todas as animacións están deseñadas para ser:

  • Suaves e naturais (0.3s de duración)
  • Non intrusivas
  • Significativas para a experiencia do usuario

4. Disposición e Espazado

O deseño segue unha grella modular con:

  • Marxes consistentes de 2rem
  • Padding harmonioso para mellorar a lexibilidade
  • Estrutura responsive que se adapta a todos os dispositivos
